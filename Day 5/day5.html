Examples of borders, padding, margins, border-radius, and background colors in CSS.

1) Borders
CSS
Borders can be solid, dotted, dashed, double, etc. Control width, style, and color. 
.border-example {
  border: 5px solid #e11d48;
  border-style: dotted; /* or dashed, double, none */
}
Rendered
Solid Border
Dotted Border
Dashed Border
Double Border
2) Padding
CSS
Padding adds space inside the element, between the content and the border. 
.padding-example {
  padding: 20px; /* all sides */
  padding-top: 40px; /* top only */
  padding-right: 40px; /* right only */
  /* etc. */
}
Rendered
Padding: 20px (all)
Top: 40px
3) Margins
CSS
Margins add space outside the element, between the border and other elements. 
.margin-example {
  margin: 20px; /* all sides */
  margin-top: 40px; /* top only */
  margin-right: 40px; /* right only */
  /* etc. */
}
Rendered
Margin: 20px
Top: 40px
4) Border-radius
CSS
Border-radius rounds the corners of an element. Use 50% for circles. 
.radius-example {
  border-radius: 10px;
  border-radius: 50%; /* circle */
  border-radius: 50px; /* pill */
}
Rendered
Rounded (10px)
Circle
Pill Shape
5) Background Colors
CSS
Set solid colors or gradients. Colors can be named, hex, rgb, hsl, etc. 
.bg-example {
  background-color: #dc2626; /* red */
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
}
Rendered
Red Background
Blue Background
Green Background
Gradient
Tip: These properties control the spacing, shape, and appearance of elements in the box model.
